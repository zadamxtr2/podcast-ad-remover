Feed Access

Control public subscription pages, RSS feed access, and feed tokens.

Public page: /subscribe. Visitors can subscribe to existing podcast feeds from this page, but cannot add, edit, reprocess, or delete anything.

User authentication is enabled. Podcast apps can use dashboard user credentials, or generated token links.

Since dashboard authentication is disabled, protected feeds need a standalone username and password.
